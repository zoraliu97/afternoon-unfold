Reading cards — batch print

Paste a JSON array below — one entry per customer, each renders as one A4 page. Click Render, then Cmd+P (or Ctrl+P) to print/save-as-PDF — A4, no margins.

Easy mode (paste from Formspree fields): name, readingText, scent, tea, colour, colourHex, boxNumber — all simple strings. Aliases accepted: customer_name, full_reading, reading_text, color, color_hex.
Optional but recommended for nicer headings: mbti (e.g. "ISFJ" → heading becomes "As ISFJ, the Defender"), zodiac ("aquarius" → "Your Aquarius heart"), todaysTarot ("The Moon"), date ("May 1" or an ISO timestamp; defaults to today).
Full mode (new orders, post-deploy): paste the reading_full_json value from each Formspree row, wrapped as an array of strings [ "json...", "json..." ]. The tool unpacks each one and uses the structured sections / today's-tarot / etc.
